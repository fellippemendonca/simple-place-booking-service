# Simple Hotel Booking Service


## Tech requirements
- Running MongoDB instance.
- config.js file shows all needed env variables.

## Setup and testing
```bash
$ npm install
```

## Start Server
```bash
$ npm start
```
## API Requests
The service contains a folder called "apiRequests" with REST-Client requests usable in VSCode.
